# DATN FarmGame Product Context

Unity 2D top-down farming/action RPG. Current priority is the playable core loop, not polish.

Core loop target:
- New game creates valid player data.
- Farm crops, harvest, receive items.
- Buy and sell through shop.
- Mine and fight for drops.
- Complete quests for rewards.
- Level up from L1 to L50, get stronger, save/load, repeat.

Defer unless explicitly assigned:
- UI polish, decorative animation, visual redesign.
- Friendship, festivals, procedural dungeons.
- Complex crafting UI, fishing, season complexity.
- Large refactors.

Every change must improve coreplay completeness, correctness, balance, testability, or reliability.
